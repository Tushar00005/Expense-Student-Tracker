# Student Expense Tracker

A full-stack student expense management dashboard designed to help users track daily spending, categorize expenses, and analyze financial habits through interactive charts. The application features secure authentication, real-time expense updates, analytics visualization, and PDF report export. Built with a modern dark-themed UI focused on usability, clarity, and professional dashboard design. The project simulates a real-world finance tracking SaaS product with responsive layout and smooth user experience.

---

## Features

- User authentication (signup/login, password visibility toggle, reset flow)
- Add, edit, and categorize expenses
- Spending analytics with interactive charts
- Necessary vs waste expense insights
- Daily spending trends
- PDF expense report export
- Responsive modern dashboard UI

---

## Tech Stack

*Frontend:* React / Vite  
*Backend:* Node.js + Express  
*Database:* MongoDB  
*Charts:* Recharts / Chart.js

---

## Installation

git clone <your-repo-link>
cd student-expense-tracker
npm install
npm run dev
